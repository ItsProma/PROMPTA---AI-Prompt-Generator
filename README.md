## ğŸ’¡ How It Works

### Example: Content Writing

**Input:**
- Category: Content Writing
- Task: Blog post about content marketing
- Tone: Professional
- Length: Medium (300-700 words)
- Audience: Marketing managers

**Generated Prompt:**
```
As an experienced content writer and SEO specialist, create a blog post about 
content marketing. Use a professional tone throughout the content. The output 
should be 300-700 words in length. Target audience: Marketing managers. 
Structure the content with clear headings, subheadings, and bullet points 
where appropriate. Include an engaging introduction and a strong conclusion.
```

## ğŸ“± Usage

### Basic Usage

1. **Select Category**
   - Choose from 7 predefined categories
   - Enter your specific task

2. **Customize**
   - Select tone, length, and language
   - Define target audience
   - Add additional context (optional)

3. **Generate**
   - Click "Generate Prompt"
   - Wait for the prompt to be created
   - Copy and use in your AI tool

### Advanced Tips

- **Be Specific**: The more details you provide, the better the prompt
- **Use Context**: Add specific requirements in the additional context field
- **Experiment**: Try different tones and lengths for optimal results
- **Save Your Prompts**: Copy and save prompts you use frequently
